API Description

For use by developers... Index is found at root "/"

Resource/Path GET POST
/sensehat/environment Current environment data (temperature and humidity) Logic in background to determine if weather conditions are good/bad Not Used
/sensehat/punctual Current time stamp (ontime/late) Logic in background to determine if employee is on time (on the hour) or late ( 3 minutes grace applied).
/sensehat/punctualpost Used to post payload 'is_ontime' with boolean True or False. Will actuate a show message on Rasb Pi with Ontime/Late
/sensehat/greenlight Used to actuate a green light on Rasb Pi when employee walks past motion sensor. (Packet Tracer) The exact time is returned to terminal.
